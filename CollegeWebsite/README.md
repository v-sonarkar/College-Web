# College Website

A React + Vite based college website project.

## Project Structure

```text
CollegeWebsite/
  fronted/
    package.json
    college_web/
      src/
      public/
      package.json
      vite.config.js
      README.md
```

## Getting Started

1. Open a terminal in `CollegeWebsite/fronted/college_web`.
2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

## Notes

- The app-specific default Vite README is available at `fronted/college_web/README.md`.
- You can replace that file later with project-specific component and page documentation.
